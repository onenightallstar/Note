## 4. Linux目录结构
- 目录规定好放什么
- 硬件也当作文件对待
- /etc 配置文件和子目录
- /usr 应用程序和文件放在此目录
- /boot linux启动的核心文件
- /tmp 存放临时文件
- /dev 设备管理器，硬件用文件的方式存储
- /media 识别设备
- /mnt 临时挂载别的文件系统
- /opt 安装软件存放目录
- /usr/local 软件安装的目标目录
- /var 存放不断扩充的，经冲修改的文件，如日志文件

## 6.关机和重启
- shutdown -h now 立即关机
- shutdown -h 1 一分钟后关机
- shutdown -r now 立即重启
- halt 关机
- reboot 重新启动
- sync把内存的数据同步到磁盘
